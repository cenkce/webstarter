# webstarter #

Web application boilerplate for front-end devlepoers.

## Installation ##

  `$ git clone git@github.com:cenkce/webstarter.git`

- Install Nodejs requirements
  `$ npm install`

- Install Bower requirements
  `$ bower install`

## Usage ##

Build project files :

  `$ gulp`
  
  or 
  
  `$ gulp --production=1`

And preview via BrowserSync :

  `$ gulp watch`
